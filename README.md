# Rubrica Contatti - Progetto JavaScript Base

Una semplice applicazione CRUD (Create, Read, Update, Delete) per la gestione di una rubrica contatti.

## Tecnologie Utilizzate
*   HTML5
*   CSS3 (con Flexbox per il layout)
*   JavaScript (ES6+)

## Funzionalità Principali
*   Visualizzazione della lista contatti tramite chiamata API (GET).
*   Aggiunta di un nuovo contatto (POST).
*   Modifica di un contatto esistente (PUT).
*   Eliminazione di un contatto (DELETE).
*   Utilizzo del `localStorage` per salvare i dati e ridurre le chiamate API.
*   Gestione asincrona delle operazioni tramite `async/await` e `fetch`.

## Scopo del Progetto
Questo progetto è stato sviluppato per mettere in pratica i concetti fondamentali di JavaScript per l'interazione con il DOM e la gestione di API REST.
